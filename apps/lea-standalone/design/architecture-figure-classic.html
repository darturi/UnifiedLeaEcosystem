Same content as the layered figure (architecture-figure.html), re-drawn in the classical icon-tile / labelled-connector idiom. The browser Web UI and the cloud Model providers sit outside the solid box — everything inside is one local process, which is why the adapter→prover edge is an in-process call, not HTTP. Dashed boxes are logical groupings; green arrows are the return path (SSE to the UI, typed events to the bridge). The Model catalog is also read by the UI (arrow omitted to reduce clutter).
